Speedtest Tracker

Port: 9020

Summary: Automated bandwidth logger using Ookla's CLI.
Purpose: Proving "congestion" (e.g., your 1Gbps line drops to 40Mbps at 8 PM).
Usage: Run 2x per hour to build a "calendar of failure" for support tickets.

Open Dashboard GitHub Source

SmokePing

Port: 9030

Summary: High-precision latency and jitter visualization.
Purpose: Identifying physical line issues (Packet Loss) vs. noise (Jitter).
Usage: Look for "smoke" (gray blur) around the lines to diagnose unstable streams.

Open Dashboard GitHub Source

Uptime Kuma

Port: 9040

Summary: Live health monitor with sub-second resolution.
Purpose: Real-time confirmation when Zoom or MS Teams flags "Unstable Connection".
Usage: Keep open on a second monitor during calls to catch "micro-outages" instantly.

Open Dashboard GitHub Source
